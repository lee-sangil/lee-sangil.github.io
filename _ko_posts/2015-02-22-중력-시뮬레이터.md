이 애플리케이션은 뉴턴의 만유인력 법칙에 따른 입자들의 운동을 시뮬레이션한다. 입자들은 서로를 끌어당기며, 충돌하여 서로 합쳐지거나 질량 중심 주위를 공전한다. 일부 입자는 우주의 지평선 너머로 사라지기도 한다. 이 애플리케이션은 대학 재학 중 반 년간의 휴학 기간 동안 직접 설계하고 구현하였다.

### 사용법
```
좌클릭: 객체 생성
드래그: 초기 속도를 가진 객체 생성
우클릭: 해당 위치에 블랙홀 생성
휠 클릭: 시점 이동
스크롤: 확대/축소
객체 위 좌클릭: 해당 객체를 화면 중심에 고정
```

### 다운로드
ID/PW: guest/admin

### 소스 코드
프로그램 종료 시 발생하는 사소한 버그를 수정한 후 곧 공개할 예정이다.
